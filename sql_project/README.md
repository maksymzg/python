# SQL Project

This is a basic SQL project with sample schema and queries.

## Files
- `schema.sql`: Database schema with table creation and sample data.
- `queries.sql`: Sample SQL queries to run on the database.

## Usage
1. Use a SQL database tool (e.g., SQLite, MySQL) to execute `schema.sql` to create the database.
2. Run queries from `queries.sql` to interact with the data.

## Troubleshooting
- Ensure you have a SQL database installed.
- For SQLite, you can run: `sqlite3 database.db < schema.sql`
